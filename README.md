## How to install
1. Clone the repo to your local storage.
``` shell
git clone https://github.com/MagicCube/movie-board.git
```
2. Install dependencies.
``` shell
npm install
```

## How to build and run
* Run in **Webpack DevServer**.
``` shell
npm start
```
* Or build it in **production mode**.
``` shell
npm run build
```
